## Overview
This project is a simple banking system using Python and Tkinter. It allows users to log in and perform common tasks like depositing money, withdrawing funds, transferring money to another account, and checking their transaction history all through a clean and interactive interface.
## Features
- User Authentication (Login system)
- Deposit and Withdraw Money
- Fund Transfer Between Accounts
- Savings Account
- Current Account 
- Transaction History Tracking
- Input Validation using Regex & Exception Handling
## Technologies Used
- Python
- Tkinter
- Regex 
## How to Run
1. Install Python on your system.
2. Install the required library: sudo apt install python3-tk
3. Run the main game file: python bank.py
